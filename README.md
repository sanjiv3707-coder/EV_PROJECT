# AI-Powered Anti-Gravity Transportation & Mobility System

An advanced engineering simulation project that explores the future of urban mobility using AI-driven anti-gravity concepts.

## 🚀 Features

- **Smart Dashboard**: Real-time monitoring of vehicle metrics.
- **Physics Simulator**: Interactive flight simulation with weight, altitude, and weather parameters.
- **ML Engine**: Predictive models for energy demand and collision risks.
- **Route AI**: 3D corridor optimization for safest and fastest travel.
- **ASTRA AI Assistant**: Chat-based system for diagnostics and control.
- **Analytics**: Comprehensive data visualization of fleet performance.
- **SQLite Integration**: Persistent logging of all simulated trips.

## 🛠️ Technology Stack

- **Frontend**: Streamlit (Futuristic UI/CSS)
- **Data Science**: NumPy, Pandas, Scikit-learn
- **Visualization**: Plotly
- **Database**: SQLite
- **AI**: Random Forest Regression Models

## ⚙️ Setup Instructions

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Initialize Data & Models**:
   Run the data generation script to train initial ML models:
   ```bash
   python data/generate_data.py
   ```

3. **Run the Application**:
   ```bash
   streamlit run main.py
   ```
