# EV Optima V2 - Premium Ground EV Optimization Dashboard

A state-of-the-art, ground-focused Electric Vehicle management and optimization system built with Next.js, Tailwind CSS, and Framer Motion.

## 🌟 Key Features

- **Premium UI/UX**: Full-screen glassmorphism dashboard with neon gradients and smooth motion transitions.
- **Street-Level Navigation**: Real-world road mapping using Leaflet.js and OpenStreetMap.
- **Smart EV Simulator**: Ground vehicle performance testing for Cars, Bikes, Scooters, Buses, and Trucks.
- **Neural Analytics**: Interactive charts (Recharts) for efficiency tracking and fleet composition.
- **Responsive Design**: Optimized for Desktop, Tablet, and Mobile devices.

## 🛠️ Technology Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Maps**: Leaflet.js / React-Leaflet
- **Charts**: Recharts

## 📂 Folder Structure

```text
src/
├── app/                  # Next.js Pages & Layouts
├── components/           # Reusable UI Components (Sidebar, Map)
├── assets/               # High-quality images & styles
├── maps/                 # Mapping logic
├── charts/               # Recharts wrappers
└── utils/                # Optimization & Physics logic
```

## ⚙️ Setup Instructions

1. **Prerequisites**: Ensure you have Node.js (v18+) installed on your machine.
2. **Navigate to the project**:
   ```bash
   cd v2_ground_ev
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Run in Development Mode**:
   ```bash
   npm run dev
   ```
5. **Open in Browser**: Navigate to `http://localhost:3000`
